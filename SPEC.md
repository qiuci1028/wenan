# 文案网站 - 设计规格文档

## 1. Concept & Vision

一个优雅的文案分享平台，收录情感、人生、励志等各类短句文案。页面追求「留白之美」，让每一句文案成为视觉焦点，给读者安静的阅读体验。整体氛围如同翻阅一本精心排版的文学杂志。

## 2. Design Language

### 美学方向
- 参考：文艺杂志 + 极简主义 + 中式留白美学
- 大量留白，内容密度低，强调文字本身的力量
- 卡片悬浮感，轻盈通透

### 配色方案
- 主色：`#1a1a1a` (墨黑 - 标题)
- 辅助色：`#666666` (灰 - 次要文字)
- 强调色：`#e8a87c` (暖橙 - 点赞/收藏高亮)
- 背景色：`#fafafa` (米白)
- 卡片色：`#ffffff` (纯白)
- 边框色：`#f0f0f0` (浅灰分隔线)

### 字体
- 主字体：`"PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", sans-serif`
- 英文/数字：`"Inter", sans-serif`
- 正文字号：16px，行高 1.8

### 空间系统
- 基础单位：8px
- 页面最大宽度：1200px
- 卡片间距：24px
- 内边距：32px

### 动效哲学
- 所有过渡 0.3s ease
- 卡片悬浮：translateY(-4px) + box-shadow 增强
- 点赞/收藏：心形/星形图标 scale 弹跳动画
- 页面切换：opacity 淡入淡出

## 4. Features & Interactions

### 核心功能

**分类浏览**
- 预设分类：全部、爱情、人生、励志、情感
- 点击分类标签筛选显示对应文案
- 当前分类高亮显示
- 切换分类时平滑过渡

**点赞功能**
- 每张卡片右下角有心形图标
- 点击切换点赞状态
- 已点赞：心形变红 + 数字+1
- 未点赞：心形灰色 + 数字不变
- 点赞数据本地存储

**收藏功能**
- 每张卡片有收藏图标（星形）
- 点击切换收藏状态
- 已收藏：星形变金黄
- 收藏数据本地存储

**我的页面**
- 展示用户收藏的文案
- 展示用户点赞的文案
- 空的 state 显示友好提示

### 交互细节
- 卡片悬浮：阴影加深，轻微上浮
- 按钮点击：scale(0.95) 按压效果
- 分类切换：下划线滑动指示
- 页面切换：淡入淡出

## 5. Component Inventory

### Header
- 固定顶部，背景白色带底部阴影
- Logo 左侧，导航居中，操作按钮右侧
- 高度 64px

### 分类标签
- 横向滚动容器
- 标签样式：padding 8px 16px，圆角 20px
- 当前选中：背景 #1a1a1a，文字白色
- 未选中：背景 #f5f5f5，文字 #666

### 文案卡片
- 白色背景，圆角 12px
- 阴影：0 2px 8px rgba(0,0,0,0.06)
- 内边距：24px
- 底部工具栏：分隔线 + 点赞/收藏按钮
- 悬浮状态：阴影加深，translateY(-4px)

### 按钮状态
- Default：灰色图标
- Hover：颜色加深
- Active：scale(0.95)
- 已激活（已点赞/已收藏）：对应强调色

## 6. Technical Approach

### 技术栈
- 纯 HTML5 + CSS3 + Vanilla JavaScript
- 无框架依赖，单文件实现
- CSS Grid + Flexbox 布局

### 数据处理
- 文案数据内嵌为 JavaScript 对象数组
- 自动分类逻辑：根据关键词匹配
- 本地存储：localStorage 保存点赞/收藏状态

### 分类算法
根据文案内容关键词自动分类：
- 爱情：爱、恋人、感情、牵手、分手、心动等
- 人生：人生、生命、时间、成长、未来等
- 励志：坚强、努力、勇敢、梦想、坚持等
- 情感：孤独、眼泪、痛苦、幸福、回忆等
